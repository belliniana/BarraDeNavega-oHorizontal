Projeto — Navbar Responsiva com Flexbox

Este projeto apresenta a construção de uma barra de navegação moderna e responsiva utilizando HTML5 e CSS3, com foco no uso prático de Flexbox para organizar e alinhar elementos de forma eficiente.

Sobre o Projeto

A página contém um cabeçalho centralizado com um logotipo e um menu horizontal totalmente estilizado. O objetivo deste projeto é demonstrar como criar uma navegação limpa, funcional e adaptável a diferentes tamanhos de tela utilizando técnicas essenciais de layout moderno.

A interface se ajusta automaticamente em telas menores, reorganizando os itens do menu com auxílio de media queries, garantindo usabilidade tanto no desktop quanto no mobile.

Tecnologias e Conceitos Abordados

HTML5 Estruturado

Uso das tags <header>, <nav>, <ul>, <li> e <a> para criar uma navegação clara e semântica.

Utilização de atributos meta para responsividade e SEO básico.

CSS3 + Flexbox

Criação de um container flexível para alinhar logotipo e menu horizontalmente.

Uso de justify-content: space-between e align-items: center para garantir organização visual.

Estilo interativo com :hover, transições suaves e destaque para o link ativo.

Aplicação de media queries para tornar o menu responsivo em telas pequenas.

Design Responsivo

Ajuste automático do menu para coluna em dispositivos móveis.

Manutenção da legibilidade e consistência estética em diferentes resoluções.

Como Executar

Baixe ou clone este repositório.

Abra o arquivo index.html em qualquer navegador moderno.

Alterações de estilo podem ser feitas no arquivo:
assets/css/style.css
